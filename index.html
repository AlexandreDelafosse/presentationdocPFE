    • Création de menu burger simple

    A l'aide de ce framework, vous pouvez créer un menu burger en quelques lignes de code.

    Installation

    Pour installer ce framework, vous devez exécuter la commande suivante : 

    npm install my-easy-menu

    Utilisation

    Importations

    Pour utiliser ce framework, vous devez importer le fichier index.js dans votre fichier HTML. 

    <script src="./node_modules/my-easy-menu/src/index.js" async></script>

    Ensuite, vous devez importer le fichier style.css dans votre fichier HTML. 

    <link href="./node_modules/my-easy-menu/src/style.css" rel="stylesheet">

    Création du menu burger

    Pour créer un menu burger, vous devez utiliser la balise menu-burger. 

    <menu-burger></menu-burger>
